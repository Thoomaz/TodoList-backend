# TuiDui - Gerenciador de Tarefas

Este repositório tem como objetivo, aprofundar os conhecimentos nos frameworks de front-end conhecidos no mercado, com o desenvolvimento deste projeto.

## Tecnologias aplicadas

- Spring Boot - Java 21 (Backend)
1. React (Front-end)
2. Vue (Front-end)
3. Angular (Front-end)

## Como executar o back-end

Vá ate o diretorio `./backend/`

E execute:

```sh
mvn spring-boot:rum
```

## Como executar cada front-end

- ### React ou Vue

**1. Instalar as dependências**

No diretório `./react/` ou `./vue/` execute o comando:

```sh
npm install
```

**2. Subir Front-end**

Para rodar o front-end, execute:

```sh
npm run dev
```
- ### Angular

**1. Executar Front-end**

No diretório `./angular/tuidui/` execute o comando:

```sh
ng server
```

Assim, o front end será compilado e poderá ser exibido.
